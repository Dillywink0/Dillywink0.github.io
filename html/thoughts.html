CD to the last working directory

cd can cd to the last working directory using a - in it. pretty neat if your moving between projects. I also learned that eshell takes this up a notch. You can use cd = to get a stack of all the directories you have visited as well as there place in the stack. You can then jump to these using cd -<number> pretty neat
